## Start Guide

### 1️⃣ Run SQL Server (Docker)
First, start the Microsoft SQL Server container:

```bash
docker run -e "ACCEPT_EULA=Y" -e "MSSQL_SA_PASSWORD=TaxiSecretPassword123@" -p 1433:1433 --name taxi-db -d [mcr.microsoft.com/mssql/server:2022-latest](https://mcr.microsoft.com/mssql/server:2022-latest)
```

### Create the Database
